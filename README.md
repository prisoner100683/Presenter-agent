## 🎯 项目概述

**Awesome Website Presenter Agent** 是一个工程化的内容转化工具，专门将演讲稿、PPT、Word 文档和 Markdown 内容转化为可交互、可演讲、视觉精致的前端网站。该工具采用成熟的工程化目录组织方式，支持多版本迭代管理，确保项目可维护性和复用性。

### 核心目标

1. **内容结构化**：从源文档提炼核心叙事、章节架构、讲解目标和演示节奏
2. **网站生成**：生成"适配讲解"的交互网站，而不仅仅是静态展示
3. **视觉精致**：确保设计质量，避免模板化和通用 AI 风格
4. **工程化交付**：完整的目录结构、脚本系统、版本管理和文档

### 适用场景

- 📚 **课堂讲解**：大学课程、技能培训、知识分享
- 🚀 **路演汇报**：融资路演、项目展示、成果汇报
- 🎓 **学生活动**：班会演讲、活动宣讲、年会展示
- 💼 **企业培训**：员工培训、产品发布、品牌介绍

## 🚀 快速开始

- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）
- 用于本地预览的 Python 3.6+ 或 Node.js
- 源文件：PPT / Docx / Markdown

### 方式一：直接打开（最简单）



## 🎨 使用流程

### 1️⃣ 准备源文件

将您的演讲稿放在 `docs/source/` 目录：

```bash
docs/source/
├── 项目介绍.pptx           # ✅ 支持
├── 培训材料.docx           # ✅ 支持
├── 讲解内容.md             # ✅ 支持
└── 演讲.pdf               # ❌ 不支持（先转为 PPTX）
```

### 2️⃣ 定义项目参数

提供以下信息以启动转化：

- **项目名**：`class-meeting`（英文，用于目录）
- **场景**：`课堂讲解` / `路演汇报` / `活动宣讲`
- **受众**：`学生` / `老师` / `评委` / `客户`
- **风格**：`专业` / `年轻` / `科技` / `学术` / `品牌化`
- **技术栈**：`HTML/CSS/JS` / `React` / `Vue`（纯前端优先）

### 3️⃣ 生成讲解网站

系统将执行以下步骤：

```
解析源文件
    ↓
提炼内容大纲与讲解节奏
    ↓
定义视觉方向（配色、字体、动效）
    ↓
搭建项目结构（目录、脚本、模板）
    ↓
生成网站代码（HTML/CSS/JS）
    ↓
响应式与可访问性测试
    ↓
输出到 output/archive/{项目名}/v1/
```

### 4️⃣ 预览与演讲

```bash
# 启动本地预览
cd output/archive/{项目名}/v1/site/
python -m http.server 8000

# 打开 http://localhost:8000
# 按 F 全屏 → 按 → 翻页 → 开始演讲
```

### 5️⃣ 迭代与发布

- 修改内容 → 生成 `v2/` 版本
- 调整设计 → 生成 `v3/` 版本
- 每个版本有独立的 `notes.md` 记录变更
- 发布时打包至 `output/release/`

## ⚙️ 配置与定制

### 修改配色方案
编辑 `site/src/styles/theme.css`：
```css
:root {
  --color-primary: #YOUR_HEX;      /* 主色 */
  --color-secondary: #YOUR_HEX;    /* 辅助色 */
  --color-gold: #YOUR_HEX;         /* 亮点 */
}
```

### 添加新幻灯片
在 `site/index.html` 的 `<div class="slides">` 中插入：
```html
<section class="slide slide--content">
  <div class="slide-content">
    <div class="title-block"><h2>你的标题</h2></div>
    <!-- 你的内容 -->
  </div>
</section>
```

### 调整动画速度
编辑 `site/src/styles/animations.css`：
```css
.slide {
  transition: opacity 0.6s;  /* 改为 0.3s 加速 或 1s 放慢 */
}
```

## 🛠️ 故障排除

### 问题 1：按键不响应
**症状**：快捷键无法工作  
**解决**：点击页面内容区域以获得焦点（不是地址栏）

### 问题 2：动画卡顿
**症状**：页面切换不流畅  
**解决**：
- 关闭浏览器扩展
- 更新显卡驱动
- 尝试其他浏览器

### 问题 3：字体显示异常
**症状**：中文显示为方块或乱码  
**解决**：
- 确认操作系统字体已安装
- 更新浏览器到最新版本
- 清除浏览器缓存（Ctrl+Shift+Delete）

### 问题 4：全屏模式不工作
**症状**：按 F 无反应  
**解决**：
- 尝试 F11 键（浏览器全屏）
- 确认浏览器允许全屏（地址栏提示）
