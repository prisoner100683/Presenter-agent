# Cache Presentation Website

A static presentation website generated from PPTX content on cache memory systems.

## How to Open and Present

1. **Open the presentation:**
   - Double-click `index.html` in the project root folder
   - Or open it in your web browser

2. **Navigation controls:**
   - **Keyboard:**
     - `ArrowRight` or `Space`: Next slide
     - `ArrowLeft`: Previous slide
     - `Home`: First slide
     - `End`: Last slide
     - `F`: Toggle fullscreen
     - `Esc`: Exit fullscreen
   - **Mouse/Touch:**
     - Click navigation dots in footer
     - Use control buttons in header

3. **Presentation features:**
   - Progress bar shows current position
   - Slide counter in header
   - Fullscreen mode for distraction-free viewing
   - Responsive design for all screen sizes
   - Accessibility features for keyboard navigation

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No external dependencies required
- Works offline

## Customization

Edit the following files to customize:
- `src/styles/theme.css`: Change colors, fonts, spacing
- `src/styles/app.css`: Modify layout and components
- `src/script.js`: Adjust presentation behavior
